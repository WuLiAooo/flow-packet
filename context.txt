• 基于当前代码，我不建议继续把 Gc “塞进”现有 Cg 节点里做隐式等待。更稳的方案是把 Gc 明确建成一种“等待/接收节点”。

  现状
  现在项目的执行模型有三个硬约束：

  - apps/server/internal/engine/runner.go 里是“节点串行执行”，而且每个节点默认是 发送一次 -> 等一个响应 -> 解码 -> 成功/失败。
  - apps/server/internal/engine/context.go 只维护按 seq 匹配的等待通道，收包兜底也只是 ResolveFirst，不适合一个 Cg 后面连续吃多个 Gc。
  - apps/server/internal/api/handler.go 和 apps/renderer/src/stores/protoStore.ts 里的路由映射只有一个 responseMsg，天然只支持 1 个请求 -> 1 个响应消
    息名。
  - apps/renderer/src/components/canvas/nodes/RequestNode.tsx 现在画布展示的是 responseFields，但这个字段实际上是在拖入节点时从当前 Cg 自己的
    message.Fields 填进去的，不是实际 Gc 返回值。

  所以按现状，Cg -> Gc -> Cg、尤其是 Cg -> Gc -> Gc -> Cg，都不是简单补一个字段能解决的。

  推荐方案
  推荐引入一个新节点类型：GcWaitNode。

  执行语义改成：

  - RequestNode：只负责发 Cg。
  - GcWaitNode：只负责等待一个指定 Gc，收到后解码并把结果挂到这个节点。
  - 连线继续保持当前线性模型，所以天然支持：
      - Cg -> GcWait -> Cg
      - Cg -> GcWait -> GcWait -> Cg

  这比“一个 Cg 节点内部配置多个 Gc 等待规则”更适合你当前项目，原因很直接：
  - 用户在画布上看得见链路，和你的使用心智一致。
  - 多个 Gc 不需要把配置做成复杂数组，直接串多个等待节点即可。
  - 每个 Gc 的返回数据有独立归属，展示和排查都更清晰。

  关键改造点
  后端要改两层：

  - 在 apps/server/internal/engine/runner.go 里给 FlowNode 增加节点类型，比如 request / wait_response。
      - wait_response 节点按 messageName 或 route/stringRoute 等待匹配包。
      - 收到包后先解码成统一结构，再分发给当前等待节点。
  - apps/server/cmd/flow-packet/main.go 的 onReceive 不能再只做 Resolve/ResolveFirst，要把未命中的包送进 runner 的通用 inbox。

  前端要改三层：

  - 在 apps/renderer/src/components/canvas/FlowCanvas.tsx 增加 GcWaitNode 类型和拖拽/创建能力。
  - 在 apps/renderer/src/stores/executionStore.ts 增加 nodeOutputs，把每个节点最近一次收到的实际数据存下来，不要塞进 canvasStore，避免污染保存的流程
    图。

  - 一个 Cg 后面可以挂多个 Gc
  - 但只有一个 Gc 允许继续连到下一个 Cg
  - 其他 Gc 只是“观察节点”，只负责展示返回数据，不参与推进流程

  - 不需要做“哪个 Gc 先回来就切哪条分支”的抢占逻辑
  - 不需要处理多个可继续分支之间的竞争、取消、仲裁

  1. 发送一个 Cg

  3. 其中“主 Gc”命中时，继续执行下一个 Cg
  4. 其他 Gc 无论什么时候回来，都只更新画布显示和日志

  - observe-only Gc 不允许再连出 Cg

  这样前后端都更好做，改动量会从“中等偏大”降到“中等”。

    154 +       plan, err := resolveExecutionPlan(nodes, edges)

    160 +
    161 +func resolveExecutionPlan(nodes []FlowNode, edges []FlowEdge) (*executionPlan, error) {
    162 +       if len(nodes) == 0 {
    163 +               return nil, fmt.Errorf("empty node list")
    164 +       }
    165 +
    166 +       nodeMap := make(map[string]*FlowNode)
    167 +       incoming := make(map[string][]string)
    168 +       outgoing := make(map[string][]string)
    169 +
    170 +       for i := range nodes {
    171 +               nodeType := normalizeNodeType(nodes[i].Type)
    172 +               if nodeType == FlowNodeTypeComment || strings.HasPrefix(nodes[i].ID, "comment") {
    173 +                       continue
    174 +               }
    175 +               nodeMap[nodes[i].ID] = &nodes[i]
    176 +               incoming[nodes[i].ID] = nil
    177 +               outgoing[nodes[i].ID] = nil
    178 +       }
    179 +
    180 +       if len(nodeMap) == 0 {
    181 +               return nil, fmt.Errorf("empty node list")
    182 +       }
    183 +
    184 +       for _, e := range edges {
    185 +               source := nodeMap[e.Source]
    186 +               target := nodeMap[e.Target]
    187 +               if source == nil || target == nil {
    188 +                       continue
    189 +               }
    190 +               outgoing[e.Source] = append(outgoing[e.Source], e.Target)
    191 +               incoming[e.Target] = append(incoming[e.Target], e.Source)
    192 +       }
    193 +
    194 +       observerWaitIDs := make(map[string]bool)
    195 +       observerWaitsByRequest := make(map[string][]string)
    196 +       requestHasAttachedWait := make(map[string]bool)
    197 +       nextNodeByID := make(map[string]string)
    198 +
    199 +       for id, node := range nodeMap {
    200 +               nodeType := normalizeNodeType(node.Type)
    201 +               switch nodeType {
    202 +               case FlowNodeTypeRequest:
    203 +                       var directRequestTargets []string
    204 +                       var continuingWaitTargets []string
    205 +                       var observerWaitTargets []string
    206 +
    207 +                       for _, targetID := range outgoing[id] {
    208 +                               target := nodeMap[targetID]
    209 +                               if target == nil {
    210 +                                       continue
    211 +                               }
    212 +
    213 +                               switch normalizeNodeType(target.Type) {
    214 +                               case FlowNodeTypeRequest:
    215 +                                       directRequestTargets = append(directRequestTargets, targetID)
    216 +                               case FlowNodeTypeWaitResponse:
    217 +                                       waitOutgoing := outgoing[targetID]
    218 +                                       if len(waitOutgoing) > 1 {
    219 +                                               return nil, fmt.Errorf("wait node %s can only continue to one request node", targetID)
    220 +                                       }
    221 +                                       if len(waitOutgoing) == 0 {
    222 +                                               observerWaitTargets = append(observerWaitTargets, targetID)
    223 +                                               continue
    224 +                                       }
    225 +
    226 +                                       nextID := waitOutgoing[0]
    227 +                                       next := nodeMap[nextID]
    228 +                                       if next == nil || normalizeNodeType(next.Type) != FlowNodeTypeRequest {
    229 +                                               return nil, fmt.Errorf("wait node %s must continue to a request node", targetID)
    230 +                                       }
    231 +                                       continuingWaitTargets = append(continuingWaitTargets, targetID)
    232 +                               default:
    233 +                                       return nil, fmt.Errorf("request node %s cannot connect to node %s", id, targetID)
    234 +                               }
    235 +                       }
    236 +
    237 +                       if len(directRequestTargets) > 1 {
    238 +                               return nil, fmt.Errorf("request node %s can only continue to one request node", id)
    239 +                       }
    240 +                       if len(continuingWaitTargets) > 1 {
    241 +                               return nil, fmt.Errorf("request node %s can only have one continuing wait node", id)
    242 +                       }
    243 +                       if len(directRequestTargets) > 0 && len(continuingWaitTargets) > 0 {
    244 +                               return nil, fmt.Errorf("request node %s has multiple continuation paths", id)
    245 +                       }
    246 +
    247 +                       if len(observerWaitTargets) > 0 || len(continuingWaitTargets) > 0 {
    248 +                               requestHasAttachedWait[id] = true
    249 +                       }
    250 +
    251 +                       if len(observerWaitTargets) > 0 {
    252 +                               observerWaitsByRequest[id] = append([]string(nil), observerWaitTargets...)
    253 +                               for _, observerID := range observerWaitTargets {
    254 +                                       observerWaitIDs[observerID] = true
    255 +                               }
    256 +                       }
    257 +
    258 +                       if len(directRequestTargets) == 1 {
    259 +                               nextNodeByID[id] = directRequestTargets[0]
    260 +                       }
    261 +                       if len(continuingWaitTargets) == 1 {
    262 +                               nextNodeByID[id] = continuingWaitTargets[0]
    263 +                       }
    264 +
    265 +               case FlowNodeTypeWaitResponse:
    266 +                       if len(outgoing[id]) > 1 {
    267 +                               return nil, fmt.Errorf("wait node %s can only continue to one request node", id)
    268 +                       }
    269 +                       if len(incoming[id]) > 1 {
    270 +                               return nil, fmt.Errorf("wait node %s can only have one parent request node", id)
    271 +                       }
    272 +                       if len(incoming[id]) == 1 {
    273 +                               parent := nodeMap[incoming[id][0]]
    274 +                               if parent == nil || normalizeNodeType(parent.Type) != FlowNodeTypeRequest {
    275 +                                       return nil, fmt.Errorf("wait node %s must be attached to a request node", id)
    276 +                               }
    277 +                       }
    278 +                       if len(outgoing[id]) == 1 {
    279 +                               nextID := outgoing[id][0]
    280 +                               next := nodeMap[nextID]
    281 +                               if next == nil || normalizeNodeType(next.Type) != FlowNodeTypeRequest {
    282 +                                       return nil, fmt.Errorf("wait node %s must continue to a request node", id)
    283 +                               }
    284 +                               nextNodeByID[id] = nextID
    285 +                       }
    286 +               }
    287 +       }
    288 +
    289 +       mainNodeIDs := make(map[string]bool)
    290 +       inDegree := make(map[string]int)
    291 +       for id := range nodeMap {
    292 +               if observerWaitIDs[id] {
    293 +                       continue
    294 +               }
    295 +               mainNodeIDs[id] = true
    296 +               inDegree[id] = 0
    297 +       }
    298 +
    299 +       for sourceID, targetID := range nextNodeByID {
    300 +               if !mainNodeIDs[sourceID] || !mainNodeIDs[targetID] {
    301 +                       continue
    302 +               }
    303 +               inDegree[targetID]++
    304 +               if inDegree[targetID] > 1 {
    305 +                       return nil, fmt.Errorf("node %s has multiple incoming execution paths", targetID)
    306 +               }
    307 +       }
    308 +
    309 +       var starts []string
    310 +       for id, degree := range inDegree {
    311 +               if degree == 0 {
    312 +                       starts = append(starts, id)
    313 +               }
    314 +       }
    315 +
    316 +       if len(starts) == 0 {
    317 +               return nil, fmt.Errorf("no start node found (cycle detected)")
    318 +       }
    319 +       if len(starts) > 1 {
    320 +               return nil, fmt.Errorf("multiple start nodes: %v", starts)
    321 +       }
    322 +
    323 +       order := make([]string, 0, len(mainNodeIDs))
    324 +       visited := make(map[string]bool)
    325 +       current := starts[0]
    326 +
    327 +       for current != "" {
    328 +               if visited[current] {
    329 +                       return nil, fmt.Errorf("cycle detected at node %s", current)
    330 +               }
    331 +               visited[current] = true
    332 +               order = append(order, current)
    333 +               current = nextNodeByID[current]
    334 +       }
    335 +
    336 +       if len(order) != len(mainNodeIDs) {
    337 +               return nil, fmt.Errorf("disconnected graph: resolved %d of %d nodes", len(order), len(mainNodeIDs))
    338 +       }
    339 +
    340 +       return &executionPlan{
    341 +               order:                  order,
    342 +               observerWaitsByRequest: observerWaitsByRequest,
    343 +               requestHasAttachedWait: requestHasAttachedWait,
    344 +       }, nil
    345 +}

    347 +func (r *Runner) Execute(ctx context.Context, nodes []FlowNode, edges []FlowEdge, onNode NodeCallback) (err error) {
    348 +       plan, err := resolveExecutionPlan(nodes, edges)
    349 +       if err != nil {
    350 +               return err
    351 +       }
    352 +
    353 +       nodeMap := make(map[string]*FlowNode)
    354 +       for i := range nodes {
    355 +               nodeMap[nodes[i].ID] = &nodes[i]
    356 +       }
    357 +
    358 +       r.mu.Lock()
    359 +       if r.running {
    360 +               r.mu.Unlock()
    361 +               return fmt.Errorf("already running")
    362 +       }
    363 +       r.running = true
    364 +       execCtx, cancel := context.WithCancel(ctx)
    365 +       r.cancel = cancel
    366 +       r.seqCtx.Reset()
    367 +       r.resetIncomingPackets()
    368 +       r.resetObserverWaits()
    369 +       r.setNodeCallback(onNode)
    370 +       r.mu.Unlock()
    371 +
    372 +       defer func() {
    373 +               r.mu.Lock()
    374 +               r.running = false
    375 +               r.cancel = nil
    376 +               r.mu.Unlock()
    377 +
    378 +               if err != nil {
    379 +                       r.resetObserverWaits()
    380 +                       r.setNodeCallback(nil)
    381 +                       return
    382 +               }
    383 +
    384 +               if !r.hasActiveObserverWaits() {
    385 +                       r.setNodeCallback(nil)
    386 +               }
    387 +       }()
    388 +
    389 +       for index, nodeID := range plan.order {
    390 +               select {
    391 +               case <-execCtx.Done():
    392 +                       err = execCtx.Err()
    393 +                       return err
    394 +               default:
    395 +               }
    396 +
    397 +               node := nodeMap[nodeID]
    398 +               if node == nil {
    399 +                       err = fmt.Errorf("node %s not found", nodeID)
    400 +                       return err
    401 +               }
    402 +
    403 +               if normalizeNodeType(node.Type) == FlowNodeTypeRequest {
    404 +                       for _, observerID := range plan.observerWaitsByRequest[nodeID] {
    405 +                               observerNode := nodeMap[observerID]
    406 +                               if observerNode != nil {
    407 +                                       r.activateObserverWait(observerNode)
    408 +                               }
    409 +                       }
    410 +               }
    411 +
    412 +               var nextNode *FlowNode
    413 +               if index+1 < len(plan.order) {
    414 +                       nextNode = nodeMap[plan.order[index+1]]
    415 +               }
    416 +
    417 +               result := r.executeNode(execCtx, node, nextNode, plan.requestHasAttachedWait[nodeID])
    418 +               r.emitNodeResult(result)
    419 +               if !result.Success {
    420 +                       err = fmt.Errorf("node %s failed: %s", nodeID, result.Error)
    421 +                       return err
    422 +               }
    423 +       }
    424 +
    425 +       return nil
    426 +}

    428 +func (r *Runner) Stop() {
    429 +       r.mu.Lock()
    430 +       defer r.mu.Unlock()
    431 +       if r.cancel != nil {
    432 +               r.cancel()
    433 +       }
    434 +       r.resetObserverWaits()
    435 +       r.setNodeCallback(nil)
    436 +}

    440 +               return
    441 +       }

    443 +       clone := &codec.Packet{
    444 +               Heartbeat:   pkt.Heartbeat,
    445 +               ExtCode:     pkt.ExtCode,

    447 +               Seq:         pkt.Seq,

    461 +       case <-r.inboxSignal:
    462 +       default:

    466 +       case r.inboxSignal <- struct{}{}:
    467 +       default:
    468 +       }

    471 +func (r *Runner) executeNode(ctx context.Context, node *FlowNode, nextNode *FlowNode, hasAttachedWait bool) NodeResult {
    472 +       switch normalizeNodeType(node.Type) {
    473 +       case FlowNodeTypeWaitResponse:
    474 +               return r.executeWaitResponseNode(ctx, node)
    475 +       case FlowNodeTypeComment:
    476 +               return NodeResult{
    477 +                       NodeID:   node.ID,
    478 +                       NodeType: FlowNodeTypeComment,
    479 +                       Success:  true,
    480 +               }
    481 +       default:
    482 +               waitForResponse := !hasAttachedWait && normalizeNodeType(getNodeType(nextNode)) != FlowNodeTypeWaitResponse
    483 +               return r.executeRequestNode(ctx, node, waitForResponse)
    484 +       }
    485 +}

    487 +func (r *Runner) executeRequestNode(ctx context.Context, node *FlowNode, waitForResponse bool) NodeResult {
    488 +       start := time.Now()
    489 +       result := NodeResult{
    490 +               NodeID:     node.ID,
    491 +               NodeType:   FlowNodeTypeRequest,
    492 +               RequestMsg: node.MessageName,
    493 +               Request:    node.Fields,
    494 +       }
    495 +
    496 +       if r.encoder == nil {
    497 +               result.Error = "message encoder not configured"
    498 +               result.Duration = time.Since(start).Milliseconds()
    499 +               return result
    500 +       }
    501 +       if waitForResponse && r.decoder == nil {
    502 +               result.Error = "message decoder not configured"
    503 +               result.Duration = time.Since(start).Milliseconds()
    504 +               return result
    505 +       }
    506 +
    507 +       payload, err := r.encoder(node.MessageName, node.Fields)
    508 +       if err != nil {
    509 +               result.Error = fmt.Sprintf("encode: %v", err)
    510 +               result.Duration = time.Since(start).Milliseconds()
    511 +               return result
    512 +       }
    513 +
    514 +       var (
    515 +               seq    uint32
    516 +               respCh chan []byte
    517 +       )
    518 +       if waitForResponse {
    519 +               seq, respCh = r.seqCtx.NextSeq()
    520 +       } else {
    521 +               seq = r.seqCtx.NextSeqValue()
    522 +       }
    523 +
    524 +       pkt := &codec.Packet{
    525 +               Route:       node.Route,
    526 +               Seq:         seq,
    527 +               Data:        payload,
    528 +               StringRoute: node.StringRoute,
    529 +       }
    530 +
    531 +       frame, err := codec.Encode(pkt, r.packetCfg)
    532 +       if err != nil {
    533 +               result.Error = fmt.Sprintf("frame encode: %v", err)
    534 +               result.Duration = time.Since(start).Milliseconds()
    535 +               return result
    536 +       }
    537 +
    538 +       if r.sendFn == nil {
    539 +               result.Error = "send function not configured"
    540 +               result.Duration = time.Since(start).Milliseconds()
    541 +               return result
    542 +       }
    543 +       if err := r.sendFn(frame); err != nil {
    544 +               result.Error = fmt.Sprintf("send: %v", err)
    545 +               result.Duration = time.Since(start).Milliseconds()
    546 +               return result
    547 +       }
    548 +
    549 +       if !waitForResponse {
    550 +               result.Success = true
    551 +               result.Duration = time.Since(start).Milliseconds()
    552 +               return result
    553 +       }
    554 +
    555 +       respData, err := r.seqCtx.WaitResponseContext(ctx, respCh, r.timeout)
    556 +       if err != nil {
    557 +               result.Error = fmt.Sprintf("wait response: %v", err)
    558 +               result.Duration = time.Since(start).Milliseconds()
    559 +               return result
    560 +       }
    561 +
    562 +       responseName := r.resolveExpectedResponseName(node.Route, node.StringRoute)
    563 +       if responseName != "" {
    564 +               result.ResponseMsg = responseName
    565 +       }
    566 +
    567 +       response, err := r.decoder(responseName, respData)
    568 +       if err != nil {
    569 +               result.Error = fmt.Sprintf("decode response: %v", err)
    570 +               result.Duration = time.Since(start).Milliseconds()
    571 +               return result
    572 +       }
    573 +
    574 +       result.Success = true
    575 +       result.Response = response
    576 +       result.Duration = time.Since(start).Milliseconds()
    577 +       return result
    578 +}

    580 +func (r *Runner) executeWaitResponseNode(ctx context.Context, node *FlowNode) NodeResult {
    581 +       start := time.Now()
    582 +       result := NodeResult{
    583 +               NodeID:      node.ID,
    584 +               NodeType:    FlowNodeTypeWaitResponse,
    585 +               ResponseMsg: node.MessageName,
    586 +       }
    587 +
    588 +       if r.decoder == nil {
    589 +               result.Error = "message decoder not configured"
    590 +               result.Duration = time.Since(start).Milliseconds()
    591 +               return result
    592 +       }
    593 +
    594 +       pkt, err := r.waitForIncomingPacket(ctx, node)
    595 +       if err != nil {
    596 +               result.Error = fmt.Sprintf("wait response: %v", err)
    597 +               result.Duration = time.Since(start).Milliseconds()
    598 +               return result
    599 +       }
    600 +
    601 +       return r.decodeWaitPacket(node, pkt, start)
    602 +}

    604 +func (r *Runner) waitForIncomingPacket(ctx context.Context, node *FlowNode) (*codec.Packet, error) {
    605 +       deadline := time.Now().Add(r.timeout)
    606 +       for {
    607 +               if pkt := r.takeMatchingPacket(node); pkt != nil {
    608 +                       return pkt, nil
    609 +               }
    610 +
    611 +               remaining := time.Until(deadline)
    612 +               if remaining <= 0 {
    613 +                       return nil, fmt.Errorf("response timeout")
    614 +               }
    615 +
    616 +               timer := time.NewTimer(remaining)
    617 +               select {
    618 +               case <-ctx.Done():
    619 +                       if !timer.Stop() {
    620 +                               <-timer.C
    621 +                       }
    622 +                       return nil, ctx.Err()
    623 +               case <-timer.C:
    624 +                       return nil, fmt.Errorf("response timeout")
    625 +               case <-r.inboxSignal:
    626 +                       if !timer.Stop() {
    627 +                               <-timer.C
    628 +                       }
    629 +               }
    630 +       }
    631 +}

    633 +func (r *Runner) takeMatchingPacket(node *FlowNode) *codec.Packet {
    634 +       r.inboxMu.Lock()
    635 +       defer r.inboxMu.Unlock()
    636 +
    637 +       for index, pkt := range r.inbox {
    638 +               if !r.matchesWaitNode(node, pkt) {
    639 +                       continue
    640 +               }
    641 +               r.inbox = append(r.inbox[:index], r.inbox[index+1:]...)
    642 +               return pkt
    643 +       }
    644 +       return nil
    645 +}

    647 +func (r *Runner) matchesWaitNode(node *FlowNode, pkt *codec.Packet) bool {
    648 +       if pkt == nil {
    649 +               return false
    650 +       }
    651 +
    652 +       hasMatcher := false
    653 +       if node.StringRoute != "" {
    654 +               hasMatcher = true
    655 +               if pkt.StringRoute != node.StringRoute {
    656 +                       return false
    657 +               }
    658 +       }
    659 +       if node.Route != 0 {
    660 +               hasMatcher = true
    661 +               if pkt.Route != node.Route {
    662 +                       return false
    663 +               }
    664 +       }
    665 +       if node.MessageName != "" {
    666 +               hasMatcher = true
    667 +               if r.resolveIncomingMessageName(pkt.Route, pkt.StringRoute) != node.MessageName {
    668 +                       return false
    669 +               }
    670 +       }
    671 +
    672 +       return hasMatcher
    673 +}

    678 +                       return responseName
    679 +               }
    680 +       }
    681 +       if r.responseResolver != nil {
    682 +               return r.responseResolver(route)
    683 +       }
    684 +       return ""
    685 +}

    708 +func (r *Runner) activateObserverWait(node *FlowNode) {
    709 +       if node == nil {
    710 +               return
    711 +       }
    712 +
    713 +       clone := *node
    714 +       r.observerMu.Lock()
    715 +       r.observerWaits[node.ID] = &clone
    716 +       r.observerMu.Unlock()

    719 +func (r *Runner) dispatchObserverPacket(pkt *codec.Packet) {
    720 +       r.observerMu.Lock()
    721 +       matched := make([]*FlowNode, 0)
    722 +       for id, node := range r.observerWaits {
    723 +               if !r.matchesWaitNode(node, pkt) {
    724 +                       continue
    725 +               }
    726 +               delete(r.observerWaits, id)
    727 +               matched = append(matched, node)
    728 +       }
    729 +       r.observerMu.Unlock()
    730 +
    731 +       for _, node := range matched {
    732 +               r.emitNodeResult(r.decodeWaitPacket(node, pkt, time.Now()))
    733 +       }
    734 +
    735 +       if !r.Running() && !r.hasActiveObserverWaits() {
    736 +               r.setNodeCallback(nil)
    737 +       }
    738 +}

    740 +func (r *Runner) decodeWaitPacket(node *FlowNode, pkt *codec.Packet, start time.Time) NodeResult {
    741 +       result := NodeResult{
    742 +               NodeID:      node.ID,
    743 +               NodeType:    FlowNodeTypeWaitResponse,
    744 +               ResponseMsg: node.MessageName,
    745 +       }
    746 +
    747 +       if r.decoder == nil {
    748 +               result.Error = "message decoder not configured"
    749 +               result.Duration = time.Since(start).Milliseconds()
    750 +               return result
    751 +       }
    752 +
    753 +       responseName := r.resolveIncomingMessageName(pkt.Route, pkt.StringRoute)
    754 +       if responseName == "" {
    755 +               responseName = node.MessageName
    756 +       }
    757 +       if responseName == "" {
    758 +               result.Error = "response message name is empty"
    759 +               result.Duration = time.Since(start).Milliseconds()
    760 +               return result
    761 +       }
    762 +       result.ResponseMsg = responseName
    763 +
    764 +       response, err := r.decoder(responseName, pkt.Data)
    765 +       if err != nil {
    766 +               result.Error = fmt.Sprintf("decode response: %v", err)
    767 +               result.Duration = time.Since(start).Milliseconds()
    768 +               return result
    769 +       }
    770 +
    771 +       result.Success = true
    772 +       result.Response = response
    773 +       result.Duration = time.Since(start).Milliseconds()
    774 +       return result
    775 +}

  对比一下这两个方案：

  1. BeginNode
     优点：

  - 执行入口是显式的，画布一眼能看出来“当前跑哪条链”
  - 配置会随画布保存，不是临时状态
  - 和你现在“点右上角 Run 就执行”这套交互兼容，不需要再加新的运行入口
  - 后端改动更小，因为本质还是“从一个起点出发执行一条链”
  - 多条独立链可以共存在画布里，但只有 BeginNode 连到的那条会参与执行，语义非常清楚

  缺点：

  - 画布上会多一个固定节点
  - 切换执行链时，要改一下 BeginNode 的连线
     优点：

  - 更灵活
  - 画布更干净

  缺点：
  - 状态不持久，别人打开这个图不一定知道上次准备跑哪条链
  - 右键运行是隐藏交互，学习成本更高
  - 选“哪一条链”这件事在实现上反而更模糊：是按选中的节点回溯到链头，还是按选中的边，还是按框选结果
  - 更容易出现“用户以为跑 A，实际跑了 B”的问题

  所以如果只选一个，我建议：

  - 进入画布默认生成唯一 BeginNode
  - BeginNode 不可删除
  - BeginNode 只能有一条输出边
  - BeginNode 只能连接到 Cg
  - 运行时只执行从 BeginNode 可达的那条链
  - 其他独立链允许存在，但不参与本次执行
  - BeginNode 没连接时，Run 按钮禁用或提示 “BeginNode is not connected”

  这是最稳、最符合当前项目的方案。

  - 右键某个 Cg 增加一个快捷动作：Set as Begin
  - 执行效果不是“右键直接运行”，而是自动把 BeginNode 切到这条链的起点

  这样两边都兼顾了：

  - 运行入口仍然显式、可持久化
  - 切换执行链也不会太麻烦

• 我理解的是这套规则：

  - 连接列表右上角现有的连接状态，继续只表示“这个服务器地址可连通”，它不是具体业务会话。
  - 所有后续协议都必须走该 session，并且每次发送都要带正确递增的 sendSequence。
  - 每个画布右上角再加一个小浮窗，显示当前 BeginNode.deviceId 对应业务 session 的状态。

  我看了当前实现，核心差异有两个：

  - 服务端现在只有一个全局 activeClient，flow.execute 直接复用它，在 apps/server/cmd/flow-packet/main.go。
  - seq 现在由 apps/server/internal/engine/context.go 从 1 开始；但你的参考客户端 C:/tool/game-test-web-client/src/main/java/com/rivergame/net/
    SocketClient.java 是从 Short.MAX_VALUE - 20 起步再自增，这里如果不对齐，后面联调大概率会直接被服端拒绝。

  我的开发计划建议按这 4 步走：

  1. 先改服务端运行时。
     把单一 activeClient 改成“探活连接 + 业务 session 池”两层。探活连接保留给当前连接页状态显示；业务 session 池按 connectionId + deviceId 缓存，并
为
     每个 session 单独维护连接对象、登录态、sendSequence、最后错误。
  2. 再给 BeginNode 加 deviceId。
     在 apps/renderer/src/stores/canvasStore.ts 扩展数据结构，在 apps/renderer/src/components/canvas/nodes/BeginNode.tsx 和属性编辑区补输入与展示；运
     行前校验 BeginNode.deviceId 必填，否则不给跑。
  3. 接着改执行入口。
     在 apps/renderer/src/components/layout/Toolbar.tsx 和后端 flow.execute 入参里把 deviceId 一并传下去。执行前先 ensureSession(deviceId)：不存在就
     建连并自动登录，存在且可用就直接复用，然后整条链都走该业务 session。
  4. 最后补状态展示和联调。
     加“当前画布 deviceId 会话状态”浮窗，状态至少包含 未配置 / 创建中 / 已连接 / 登录中 / 已登录 / 失败 / 已断开。同时验证重复运行、断线后重建、切
换
     不同 tab 不同 deviceId 的行为。

  我建议就按这个顺序做，因为第 1 步是基础，不先改它，前端 BeginNode.deviceId 做出来也跑不通。

  我这里只有一个需要你确认的点：
  sendSequence 我是否直接完全对齐参考客户端，按 Short.MAX_VALUE - 20 作为每个新业务 session 的初始值来实现？如果你确认，我就从第 1 步开始落地。
